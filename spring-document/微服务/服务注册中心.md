#### Nacos

AP+CP架构



| 功能特性       | Zookeeper          | Eureka            | Nacos                      |
| -------------- | ------------------ | ----------------- | -------------------------- |
| 一致性协议     | CP                 | AP                | CP+AP                      |
| 健康检查       | Keep Alive(长连接) | Client Beat(ping) | TCP/HTTP/MYSQL/Client Beat |
| 负载均衡策略   | —                  | Ribbon            | 权重/metadata/Selector     |
| 雪崩保护       | 无                 | 有                | 有                         |
| 访问协议       | TCP                | HTTP              | HTTP/DNS                   |
| 跨注册中心同步 | 不支持             | 不支持            | 支持                       |
| K8S集成        | 不支持             | 不支持            | 支持                       |
