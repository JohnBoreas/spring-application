- 什么是 Spring Messaging

  Spring Messaging 是 Spring Framework 中的一个模块，其作用就是统一消息的编程模型。

  Messaging （包括：消息体 Payload 和消息头 Header）

  MessageChannel （消息通道 ）发送消息

  SubscribableChannel （）接受消息

  MessageHandler消息处理器订阅消费
