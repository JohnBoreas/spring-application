一、LinkedList的概述

LinkedList是双向链表实现的

ListLinkedList是非线程安全的

LinkedList元素允许为null，允许重复元素

LinkedList是基于链表实现的，因此插入删除效率高，查找效率低（虽然有一个加速动作）

LinkedList是基于链表实现的，因此不存在容量不足的问题，所以没有扩容的方法

LinkedList还实现了栈和队列的操作方法，因此也可以作为栈、队列和双端队列来使用
