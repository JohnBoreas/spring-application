定时器实现方案：

1、JDK的Timer包

2、Thread

3、ScheduledThreadPool

4、Spring Timer(@Scheduled 推荐)

5、Quartz





一、Timer

单线程工作，由一个队列、一个线程组成

（1）缺点

1、多个任务，且任务时间过长，超过了两个任务的间隔时间，会导致后续任务被推迟进行

2、抛出RuntimeException，Timer会停止所有任务的运行

3、执行周期任务时依赖系统时间



二、Thread

相当于run里面进行一个死循环



三、ScheduledThreadPool

其实就是把任务交给了线程池去执行
