RabbitMQ的六种工作模式：

1、Work queues

2、Publish/subscribe（发布订阅）

3、Routing（路由）

4、Topics（会话）

5、Header 模式

6、RPC



#### 五种工作模式的主要特点

1. 简单模式：一个生产者，一个消费者
2. work模式：一个生产者，多个消费者，每个消费者获取到的消息唯一（消费者彼此竞争成为接收者）。
3. 订阅模式：一个生产者发送的消息会被多个消费者获取。
4. 路由模式：发送消息到交换机并且要指定路由key ，消费者将队列绑定到交换机时需要指定路由key
5. topic模式：将路由键和某模式进行匹配，此时队列需要绑定在一个模式上，“#”匹配一个词或多个词，“*”只匹配一个词。
