#### 1、特性

Java是跨平台的语言

Java虚拟机只关心字节码文件

#### 2、Java发展的重大事件

（1）1990年，Java问世

（2）2000年，1.3发布，`Hotspot`成`Java`默认虚拟机

（3）2006年，JDK6发布，Java开源并建立`OpenJDK`

（4）2010年，oracle收购sun，获得Hotspot

（5）2011年JDK7发布，企业新的垃圾回收器G1

（6）2017年，JDK9发布，G1成默认GC，替代CMS

#### 3、JVM发展历程

（1）Sun classic VM

1996年，世界第一款商用虚拟机（Sun classic VM），1.4时被淘汰

内部只提供解析器

（2）Exact VM

1.2上提供

（3）Hotspot VM

1.3成为默认vm

（4）JRockit VM
